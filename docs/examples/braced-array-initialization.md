# Braced array initialization {#example_braced_array_initialization}

<!-- source:braced-array-initialization.cpp -->
```{.cpp}
int main()
{
    // The compiler automatically fills the remaining
    // fields with 0 thanks to braced initialization.
    int arr[5]{2, 3, 4};
}
```
